# Completed Plans

This folder holds implementation plans that have been fully executed, verified, and archived.

## When to Archive

Move a plan here when:

- Implementation is complete
- Verification evidence exists (tests pass, manual confirmation, etc.)
- The user has confirmed or approved the closeout
- UPDATE PROCESS mode has captured any learnings

## How to Archive

1. Move the plan file from `active/` to this folder
2. Update any references to the plan path if needed
3. Optionally add a completion date to the filename or a status line in the plan

## What Does Not Go Here

- Plans still being tested (keep in `active/`)
- Plans that were abandoned (move to `backlog/` with a note, or delete)
- Reports about the implementation (use `reports/`)
